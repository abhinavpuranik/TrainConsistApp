# TrainConsistApp
Train Consist Management App is a small Java project developed to demonstrate the use of core Java concepts through a sequence of incremental use cases. The application models basic operations involved in managing a train consist, such as initializing the system, adding and organizing bogies, ensuring uniqueness of bogie IDs, maintaining ordered formations, mapping bogie capacities, sorting and filtering bogies, grouping data using streams, validating inputs using regular expressions, performing safety compliance checks, comparing performance between loops and streams, handling errors using custom exceptions, and implementing searching and sorting algorithms. Each use case introduces a different concept from the Java Collections Framework, Stream API, exception handling, and basic algorithms, gradually building a more structured understanding of how data structures and control logic can be applied in a simple train management context.
